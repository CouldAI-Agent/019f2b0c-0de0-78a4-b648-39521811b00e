# Corrugated Material Hub

A comprehensive platform designed for the corrugated box industry. This application bridges the gap between buyers and sellers of corrugated box raw materials.

## Features
*   **Marketplace Directory:** Discover and connect with reliable sellers and buyers. Contact information (phone/website) is readily available.
*   **User Registration:** Seamless onboarding process for both buyers and sellers to expand their business reach.
*   **Company Management:** A dedicated portal for organizational structure, featuring job openings and applications for critical roles like CEO, HR Operations Manager, Marketing Sales, and Operations Manager.

## Tech Stack
*   Flutter (iOS, Android, Web, Desktop)

## Getting Started
1. Ensure Flutter is installed.
2. Run `flutter pub get` to fetch dependencies.
3. Start the application with `flutter run`.
